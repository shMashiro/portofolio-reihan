CARA MENJALANKAN PROJECT

1. Clone repository
   git clone https://github.com/shMashiro/portofolio-reihan.git

2. Masuk ke folder project
   cd portofolio-reihan

3. Install dependencies
   npm install
   atau
   yarn
   atau
   pnpm install

4. Jalankan development server
   npm run dev
   atau
   yarn dev
   atau
   pnpm dev

5. Buka di browser
   http://localhost:3000
